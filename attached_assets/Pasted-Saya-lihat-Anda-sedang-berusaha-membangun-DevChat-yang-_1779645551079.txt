### 🧠 3. Ubah AI Menjadi Agen yang Cerdas

AI Anda perlu mengubah instruksi bahasa alami menjadi serangkaian tindakan. Arsitektur **Reasoning + Acting (ReAct)** adalah yang paling populer.

1.  **Loop Utama**: Sistem akan menerima instruksi, memanggil AI, dan menjalankan alat yang diminta berulang kali.
2.  **Manajemen Konteks**: Jika percakapan sudah panjang, ringkas riwayatnya agar AI tidak lupa instruksi awal.
3.  **Struktur Tools**: Definisikan alat yang tersedia untuk AI dalam format JSON.

<details>
<summary><b>💡 Contoh Kode: Loop Agen AI yang Cerdas</b></summary>

```javascript
import Anthropic from '@anthropic-ai/sdk';
import tools from './tools.js';

const anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

// Definisikan tools dalam format yang dimengerti AI
const toolDefinitions = [
  {
    name: 'read_file',
    description: 'Membaca konten file',
    input_schema: {
      type: 'object',
      properties: { path: { type: 'string', description: 'Path file' } },
      required: ['path'],
    },
  },
  {
    name: 'edit_file',
    description: 'Mengedit file dengan string replacement',
    input_schema: {
      type: 'object',
      properties: {
        path: { type: 'string' },
        old_str: { type: 'string' },
        new_str: { type: 'string' },
      },
      required: ['path', 'old_str', 'new_str'],
    },
  },
  {
    name: 'run_shell_command',
    description: 'Menjalankan perintah shell',
    input_schema: {
      type: 'object',
      properties: { command: { type: 'string' } },
      required: ['command'],
    },
  },
];

async function runAgent(userInput, conversationHistory = []) {
  let messages = [...conversationHistory, { role: 'user', content: userInput }];
  let maxIterations = 10;
  
  for (let i = 0; i < maxIterations; i++) {
    // Panggil AI dengan tools yang tersedia
    const response = await anthropic.messages.create({
      model: 'claude-3-5-sonnet-20241022',
      max_tokens: 4096,
      messages: messages,
      tools: toolDefinitions,
    });
    
    // Tambahkan respons AI ke riwayat
    messages.push(response.content[0]);
    
    // Jika AI merespon dengan teks biasa, selesai
    if (response.stop_reason !== 'tool_use') {
      return response.content[0].text;
    }
    
    // Proses setiap alat yang ingin dipanggil AI
    for (const content of response.content) {
      if (content.type === 'tool_use') {
        const toolName = content.name;
        const toolInput = content.input;
        console.log(`🤖 AI memanggil tool: ${toolName}`);
        
        // Jalankan fungsi yang sesuai
        const result = await tools[toolName](toolInput);
        
        // Kirim hasil kembali ke AI
        messages.push({
          role: 'user',
          content: [{
            type: 'tool_result',
            tool_use_id: content.id,
            content: JSON.stringify(result),
          }],
        });
      }
    }
  }
  return 'Mencapai batas maksimum iterasi';
}
```

### 🌐 4. Rancang Arsitektur Web Anda

Agar AI ini terintegrasi dengan website dan RTK yang sudah Anda install.

1.  **Backend**: Buat **server WebSocket** untuk streaming interaksi secara real-time.
2.  **Frontend**: Di React, kelola state global AI (misal: `isAgentRunning`, `isWaitingForUser`) dengan **Redux Toolkit** dan sinkronisasi data dengan **RTK Query**.
3.  **Keamanan**: Jalankan tools AI di **lingkungan yang terisolasi (sandbox)**, misalnya dengan Docker, untuk mencegah risiko keamanan.

### 🔄 Konsep Penting untuk AI Agen

*   **Restart Workflow**: Sediakan fungsi yang bisa mengatur ulang state AI ke awal.
*   **Checkpoint**: Sediakan mekanisme "titik penyimpanan" proyek agar pengguna bisa kembali ke versi sebelumnya jika perubahan AI salah.
*   **Prompt Queue**: Implementasikan antrean untuk instruksi beruntun. Ini memungkinkan AI tetap bekerja meskipun pengguna memberikan perintah baru, seperti fitur Queue di Replit.
